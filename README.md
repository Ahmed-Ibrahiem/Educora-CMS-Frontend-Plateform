# Educora - CMS Educational Platform

Educora is a modern educational CMS platform built using **React, TypeScript, and Vite**, designed with a scalable architecture that simulates real-world LMS applications.

---

## Overview

Educora is a frontend-focused CMS platform where users can explore courses, view detailed course content, and interact with a dynamic and structured user interface.

The project focuses on building a **production-like frontend architecture**, emphasizing scalability, maintainability, and clean code practices.

---

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Context API (Global State Management)
- React Router DOM
- Axios
- React Hook Form + Yup (Form Validation)
- Chart.js
- Lottie Animations
- Swiper.js

---

## Key Concepts Applied

- Scalable and modular folder structure
- Strong typing using TypeScript (Interfaces & Types)
- Global state management using Context API
- API abstraction using Axios (with local JSON data simulation)
- Form handling & validation using React Hook Form + Yup
- Clean separation between UI, logic, and services
- Reusable and maintainable components

---

## Project Structure

src/
├── Components # Reusable UI components
├── Pages # Application صفحات
├── Hooks # Custom hooks
├── Contexts # Global state management
├── Services # API & business logic
├── types # TypeScript types & interfaces
├── Utils # Helper functions
├── assets # Static assets
├── animations # Lottie animations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

```
